# Glow by Avishag — מערכת הזמנת תורים

אפליקציית הזמנת תורים לסטודיו Glow by Avishag.  
בנויה עם React, TypeScript, Vite, shadcn-ui ו-Tailwind CSS.

## התקנה והפעלה

```sh
# התקנת תלויות
npm install

# הפעלת שרת פיתוח
npm run dev

# בניית גרסת ייצור
npm run build
```

## טכנולוגיות

- **Vite** — כלי בנייה מהיר
- **React + TypeScript** — ממשק משתמש
- **shadcn-ui** — ספריית קומפוננטות
- **Tailwind CSS** — עיצוב
